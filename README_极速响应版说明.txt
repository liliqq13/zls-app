智路哨兵极速响应版

本版专门解决“实时地图和实时天气响应时间太久”的问题：

一、地图响应优化
1. 首屏不立即初始化高德地图。
2. 地图页先显示本地测试道路，车辆和障碍先本地渲染。
3. 用户进入地图页后，再后台加载高德线上地图。
4. 高德路线规划结果使用本地缓存，下次打开更快。
5. 高德路线规划超过 2.5 秒未返回时，继续显示本地路线，不阻塞 APP。
6. 不在非地图页更新高德地图，减少卡顿。

二、天气响应优化
1. 首屏先读取 localStorage 天气缓存，立即显示天气图标。
2. 和风天气后台异步刷新，不阻塞 APP。
3. 天气接口设置超时兜底，接口慢时保留缓存/测试天气。
4. 天气不再每秒请求，改为每 10 分钟刷新一次。
5. 顶部、首页、环境模块、预警中心都同步同一份实时天气状态。

三、保留功能
- 高德线上地图
- 和风实时天气
- 预警天气同步
- 行驶总长、已行驶、剩余距离
- 动态目标、静态障碍/凹陷、风险点
- 详细车辆数据
- 天气图标实时联动
