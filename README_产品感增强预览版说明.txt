智路哨兵产品感增强预览版

本版在“一键登录预警颜色版”基础上增强真实产品感：
1. 首页增加系统闭环状态：车辆在线、RTK FIX、CAN监听、天气同步、预警模块、地图兜底。
2. 预警中心增加风险评分、最近目标距离、预计碰撞时间、识别置信度、预警依据。
3. 地图页增加真实底图状态：本地实际路线、高德底图待加载/加载中/已加载/失败兜底。
4. 数据页增加接口状态与预警闭环状态。
5. 预警颜色仍保持：正常绿色、中风险黄色、高风险红色。
6. 登录保留，账号和密码已预填，只需点击登录。
