智路哨兵车主安全平台 V4.6 地图防重叠修复版

针对V4.5截图中的重叠问题完成修复：
1. 路线自动缩放到地图安全显示区，不再被底部车辆卡片遮挡。
2. 三个风险事件标签自动判断左右与上下方向，防止超出屏幕。
3. 风险标签增加连接线，标签与路线点位关系更清楚。
4. 顶部车辆切换栏隐藏横向滚动条。
5. 天气卡与本地路线提示左右分离，不再相互覆盖。
6. 地图图例上移，与底部车辆卡片保持安全距离。
7. 高德资源不可用时不再弹出“页面资源加载异常”遮挡页面。
