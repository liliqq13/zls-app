智路哨兵免登录直达版

本版专门修复“还是进入不了”的问题。

关键修复：
1. 修复登录层重复 class 导致 hide 不生效的问题。
2. 登录遮罩默认 display:none，不再挡住系统页面。
3. 登录按钮保留，但点击后直接进入。
4. 自动注销旧 Service Worker，减少 GitHub Pages 旧缓存影响。
5. 保留实时地图极速版全部功能：
   - 高德真实在线地图
   - GPS 轨迹直绘
   - 车辆实时移动
   - 绿色已行驶路线
   - 动态目标、静态障碍、风险点
   - 和风实时天气
   - 预警天气同步
   - 详细车辆数据
