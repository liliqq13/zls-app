智路哨兵设备日志导出版

新增功能：
1. 数据页新增“设备状态日志”模块。
2. 实时记录设备状态、CAN监听、RTK定位、通信链路、天气同步、地图模块、电源状态和预警事件。
3. 显示设备温度、供电电压、通信状态、日志数量。
4. 支持导出 CSV、TXT、JSON。
5. 支持清空日志。
6. 日志内容包含时间、等级、模块、消息、详情、车辆编号、天气、路线。
7. 可用于设备状态监测、故障追踪、现场验收和比赛展示。
