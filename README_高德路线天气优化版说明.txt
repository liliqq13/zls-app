智路哨兵高德路线天气优化版

本版按要求修改：
1. 增加永川实时天气预报：
   - 高德实时天气
   - 温度
   - 风向风力
   - 湿度
   - 未来预报
2. 地图显示速度优化：
   - 高德地图改为简洁模式
   - 关闭复杂标注
   - 减少频繁居中和路线重绘
   - 规划路线缓存，不重复请求
3. 按地图上的路线规划一条测试道路：
   - 使用高德驾车路线规划
   - 蓝色显示规划测试道路
   - 绿色显示已行驶路线
   - 保留车辆、动态目标、静态障碍/凹陷、风险点
4. 仍保留：
   - 行驶总长、已行驶、剩余距离
   - RTK 20厘米级定位
   - 兴龙湖大雨
   - 茶山竹海大雾
   - 预警详细车辆数据

注意：
- 高德后台需要开启 JS API、天气查询、路线规划相关服务权限。
- 域名白名单需要包含：euphonious-elf-6f5eee.netlify.app
- 如果天气或路线规划不显示，通常是服务权限、配额或域名白名单问题。

?v=amap_weather_route
