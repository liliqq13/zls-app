智路哨兵实时地图极速版

本版解决“地图需要实时地图，但响应速度太慢”的问题。

核心修改：
1. 保留高德真实在线地图。
2. 删除最耗时的高德 Driving 驾车路线规划插件。
3. 不再等待路线规划返回，直接使用车辆 GPS 轨迹绘制蓝色路线。
4. 绿色已行驶路线按车辆实时位置更新。
5. 高德地图后台预加载，进入地图页后快速显示真实底图。
6. 地图每秒更新车辆、动态目标、静态障碍、风险点。
7. 地图中心每 6 秒跟随一次车辆，避免每秒拖动地图导致卡顿。
8. 天气继续使用和风天气缓存优先机制。
9. 预警中心继续同步实时天气。

上传 GitHub Pages 或其他平台后打开：
?v=realtime_map_fast

说明：
这版是“真实在线地图 + 车辆GPS轨迹直绘”，比“真实在线地图 + 高德路线规划”快很多。
如果后续需要真实道路导航路线，可在演示稳定后再单独做“路线规划按钮”，不要默认启动。
