智路哨兵手机端地图修复版

本版根据手机截图修复地图显示问题：
1. 高德在线地图加载成功后，彻底隐藏本地 SVG 叠层，避免出现双路线、文字标签压在真实地图上。
2. 本地路线模式保留，手机端高德加载失败时仍能演示车辆沿实际路线行驶。
3. 高德在线模式下，绿色已行驶路线会包含当前车辆实时坐标，轨迹更连续。
4. 地图底部信息卡在手机端更紧凑，减少遮挡。
5. 本地路线标签限制在可视范围内，减少右侧文字被裁切。
6. 保留一键登录、LOGO、实际路线、天气全局同步、预警颜色和产品感增强功能。
