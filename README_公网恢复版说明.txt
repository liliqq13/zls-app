智路哨兵公网恢复版

本版用于解决“公网平台用不了了/页面打不开/页面初始化失败”的问题。

修复内容：
1. 修复天气同步函数中 fallback 未定义可能导致页面停止运行的问题。
2. 天气接口失败时不影响 APP 主页面。
3. 高德地图加载失败时自动保留本地路线地图。
4. renderFrame 加入兜底保护，单个模块报错不拖垮整页。
5. 保留公网演示所需功能：
   - 首页车辆状态
   - 地图页本地路线优先
   - 高德线上地图后台加载
   - 和风天气后台加载
   - 预警中心
   - 详细车辆数据
   - 行驶长度、已行驶、剩余距离

上传 Netlify 后建议使用：
?v=public_recover

如果仍然打不开：
1. 确认上传时 index.html 在根目录，不要多套一层文件夹。
2. 清除浏览器缓存。
3. 用无痕模式打开。
4. 或新建一个 Netlify 站点重新上传本 ZIP。
