智路哨兵实时天气全局同步版

本版专门修复“实时天气同步图标还有问题”的情况。

核心修改：
1. 全系统只保留一份天气状态 weather。
2. 和风天气接口成功后，车辆切换不会再被本地天气覆盖。
3. 顶部天气图标、首页大图标、天气页、预警中心、事件记录、数据页、地图角标全部使用同一份实时天气。
4. 未来 3 天天气预报同步和风天气接口，不再只显示本地预设。
5. 接口慢或失败时使用缓存/本地兜底，但界面会统一显示，不会出现不同页面天气不一致。
6. 保留登录：
   账号 operator@zls.local
   密码 123456
7. 保留已换好的智路哨兵 LOGO、实际路线行驶和手机端本地地图兜底。
