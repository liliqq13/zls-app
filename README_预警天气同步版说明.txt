智路哨兵高德线上地图预警天气同步版

本版按要求修改：
1. 预警中心同步和风实时天气。
2. 预警主卡显示实时天气、温度、湿度、风况。
3. 预警详细车辆数据里的天气卡改为实时天气。
4. 预警列表每条风险事件显示实时天气。
5. 无预警状态也显示当前预警气象源。
6. 不再使用本地模拟天气覆盖预警天气。
7. 保留：
   - 高德线上地图
   - 高德路线规划
   - 和风实时天气
   - 天气图标联动
   - 行驶长度、已行驶、剩余距离
   - 动态目标、静态障碍/凹陷、风险点
   - 详细车辆数据
