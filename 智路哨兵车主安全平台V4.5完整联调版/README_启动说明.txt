智路哨兵真实平台联调原型版
================================

这不是单纯的静态页面动画，而是一套可以在电脑本地直接运行的前后端联调原型。

一、已实现的“真实平台”能力
1. Node.js 后端 API。
2. 登录接口与访问令牌。
3. SSE 实时遥测数据流。
4. 三辆车辆、三台设备的实时状态。
5. 设备心跳在线状态动态计算。
6. 车辆—设备—协议库—标准转换—风险模型绑定链路。
7. 风险判断解释器与规则/模型/阈值版本快照。
8. 工单由后端创建、持久化和完成。
9. 报表由服务器生成，并写入报告编号、哈希和审计记录。
10. 安全审计日志。
11. 异常模拟：CAN、RTK、温度、网络、障碍物、大雾。
12. 后端不可用时自动进入离线预览模式。

二、Windows 启动
1. 电脑安装 Node.js 18 或更高版本。
2. 双击 start.bat。
3. 浏览器打开：
   http://localhost:8787

四、数据保存位置
data/db.json
- 工单
- 报表索引
- 审计日志
- 阈值配置

reports/
- 服务器生成的 JSON 报表

五、GitHub Pages 说明
GitHub Pages 只能运行静态前端，不能运行 server.js。
可以单独上传 public/index.html 进行离线预览，但工单、服务端报表、登录令牌、SSE 数据流不会是真实后端。

六、正式上线还需要继续补充
1. PostgreSQL / TimescaleDB 或其他正式数据库。
2. MQTT 设备网关和真实车载盒子数据。
3. HTTPS、设备证书、密钥轮换。
4. 后端角色权限校验。
5. 正式报表 PDF/Excel 生成。
6. 高德和天气接口后端代理。
7. OTA签名、灰度升级和回滚。
8. 容器化部署、监控、备份和安全审计。

本包定位：可运行的真实平台联调原型，不是生产级商用系统。
