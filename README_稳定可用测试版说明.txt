智路哨兵稳定可用测试版

这是一版重新整理后的稳定版本，不再依赖登录、CSV、Service Worker。

使用方式：
1. 解压 ZIP。
2. 只上传 index.html 覆盖 GitHub Pages 仓库里的 index.html。
3. 打开：https://liliqq13.github.io/ZLS-app/?v=stable_tested

已测试项目：
- 页面可打开
- 首页数据实时变化
- 底部 5 个导航可切换
- 三类车辆可切换
- 地图页可进入，在线地图慢时本地地图兜底
- 车辆轨迹实时运动
- 预警中心可显示风险事件
- 测试预警按钮可用
- 天气慢时不会卡死页面
