智路哨兵免登录修复版

本版用于解决 GitHub Pages 或公网演示时“登录不了”的问题。

修改内容：
1. 默认直接进入系统，不再卡在登录页。
2. 登录按钮仍保留，但点击后直接进入。
3. 适合比赛现场、评委扫码、公网演示。
4. 保留实时高德地图极速版功能：
   - 高德真实在线地图
   - GPS轨迹直绘
   - 车辆实时移动
   - 绿色已行驶路线
   - 动态目标、静态障碍、风险点
   - 和风实时天气
   - 预警天气同步
   - 详细车辆数据

上传 GitHub Pages 后使用：
?v=no_login_fix

正确访问示例：
https://liliqq13.github.io/zls-app/?v=no_login_fix
或：
https://liliqq13.github.io/ZLS-app/?v=no_login_fix
