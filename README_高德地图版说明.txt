智路哨兵高德地图版

本版已接入高德地图 Web JS API：
1. 地图页显示高德真实底图。
2. 蓝色路线：完整行驶路线。
3. 绿色路线：已行驶里程。
4. 地图显示公里标记。
5. 地图显示车辆实时位置。
6. 车辆接近后显示动态目标、静态障碍/凹陷、风险点。
7. 保留：
   - 行驶总长
   - 已行驶里程
   - 剩余距离
   - 能见度/湿度/雨量实时更新
   - 预警详细车辆数据
   - 兴龙湖大雨、茶山竹海大雾

注意：
- 需要高德 Key 的域名白名单包含 euphonious-elf-6f5eee.netlify.app
- 如果地图不显示，请检查 Key、jscode、域名白名单是否正确。
- 当前版本为演示版，Key 与安全密钥写在前端；正式商用建议改为后端代理方式。
